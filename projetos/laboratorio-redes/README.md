# 🌐 Laboratório de Redes

Estudos e testes práticos com comandos de rede.

## Comandos estudados

### PING
Testa conectividade entre dispositivos.

Exemplo:
ping google.com

### TRACERT
Mostra o caminho que os pacotes percorrem na rede.

Exemplo:
tracert google.com

### IPCONFIG
Mostra informações da rede do computador.

Exemplo:
ipconfig
