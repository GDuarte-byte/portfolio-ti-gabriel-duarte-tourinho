# 🔧 Manutenção de Computadores

Projeto com documentação básica de manutenção de hardware.

## Conteúdo

- Diagnóstico de falhas
- Teste de memória RAM
- Troca de HD por SSD
- Limpeza interna do computador
- Monitoramento de temperatura

## Objetivo

Demonstrar conhecimentos iniciais em manutenção e troubleshooting de computadores.
