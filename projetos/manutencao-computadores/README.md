# 🔧 Manutenção de Computadores

Projeto com documentação básica de diagnóstico e manutenção de hardware.

---

## 🎯 Objetivo

Demonstrar conhecimentos em identificação e solução de problemas comuns em computadores.

---

## 🧪 Problemas analisados

### Computador lento

Possíveis causas:

- Pouca memória RAM
- Disco rígido lento
- Muitos programas iniciando com o sistema

Soluções aplicadas:

- Verificação de uso de memória
- Desativação de programas de inicialização
- Instalação de SSD

---

### Computador não liga

Verificações realizadas:

- Fonte de alimentação
- Cabos internos
- Memória RAM

---

### Superaquecimento

Possíveis causas:

- Acúmulo de poeira
- Cooler com defeito

Solução:

- Limpeza interna do computador
- Verificação do sistema de ventilação

---

## 🛠 Ferramentas utilizadas

- BIOS
- Gerenciador de Tarefas
- Ferramentas de diagnóstico
